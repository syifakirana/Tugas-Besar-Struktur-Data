# KATALOG BUKU DAN PEMINJAMAN PERPUSTAKAAN DIGITAL

<p> Program ini merupakan program berbasis C++ yang digunakan untuk mengelola data buku perpustakaan dan peminjaman oleh member. Data buku disimpan menggunakan Binary Search Tree (BST) sehingga proses penambahan, pencarian, pembaruan, dan penghapusan buku dapat dilakukan secara terstruktur berdasarkan ISBN. Setiap buku memiliki informasi seperti judul, penulis, tahun terbit, dan tag topik.

Data peminjaman dikelola menggunakan Multiple Linked List (MLL), di mana setiap member dapat memiliki lebih dari satu buku yang dipinjam. Program ini mendukung pengelolaan data member, proses peminjaman dan pengembalian buku, serta penampilan data member beserta daftar buku yang sedang dipinjam.

Selain fungsi utama, program ini juga menyediakan fitur pengolahan data seperti menghitung jumlah buku, pencarian buku berdasarkan tag, pengolahan tahun terbit (minimum, maksimum, dan rata-rata), serta statistik aktivitas member berdasarkan jumlah peminjaman.</p>
